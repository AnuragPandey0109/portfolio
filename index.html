About

Who I am

I'm Anurag Kumar Pandey, a passionate Full Stack Web Developer with strong expertise in Java and modern frontend technologies like React, HTML, and CSS. I specialize in building scalable, high-performance web applications with clean architecture and responsive user interfaces.

I enjoy creating efficient backend systems and intuitive frontend experiences that work seamlessly together. My primary weapon for backend development is Spring Boot, which I use to implement Microservices and RESTful APIs that are secure, thread-safe, and ready for the cloud. I combine this with Node.js and MongoDB depending on the project's real-time needs.

Constantly learning and improving, I stay updated with the latest technologies and best practices in web development. My goal is to develop impactful web applications that deliver real-world value and provide excellent user experiences.
